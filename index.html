📄 PDF Stitcher

Merge PDFs offline. Fast, simple, and privacy-first.

View on GitHub

🧠 Why this exists

Most PDF tools require uploading documents online. That doesn’t work when files are sensitive.

This tool runs completely offline, in your browser, giving you full control.

🚀 What it does

✔ Add PDFs in multiple batches
✔ Drag & drop files
✔ Reorder before merging
✔ Remove unwanted files
✔ Custom output filename
✔ One-click reset
✔ Fully offline (no uploads)

⚙️ How it works

Built using Python (Flask) and pypdf, with a lightweight browser interface.

▶️ Get started

Clone the repository, run the app locally, and start merging PDFs instantly.
